# Webuntis
A API library that makes it easy to access the Webuntis JSON RPC 2.0 API

## Features

* easy to extend
* flexible
* fast through caching
* easy to use

## Getting Started 
These instructions will show you how to get this package up and running.

### Requirements

* php 5.6 or higher 
* php memcached

### Installation

```
composer require tfranek/webuntis
```
